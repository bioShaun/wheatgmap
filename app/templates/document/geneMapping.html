Gene Mapping Workflow

The typical workflow for BSA gene mapping

1. Upload your VCF

a)The VCF format should flow the GATK Best Practice or our suggest pipeline. If you try to provide vcf from other tools, please make sure that the “Format” column should be “GT:AD:DP:GQ:PL” or at least contain “GT:AD:DP”.

b)The genome for call variation pipeline must be IWGSC RefSeq V1.0.
c)The VCF file size was limited as lease than 1G. If you have large VCF file, please split into small one by each sample.
d)When you finished upload your VCF, the system will take a several minutes to process the file in the background. You can check the status by Tasks page.

2. BSA

a) Wild Bulk and Mutant Bulk are required.
b) Wild Parent and Mutant Parent are suggested if available.
c) Background could help you filter out some false positive variants, but you should be much careful the genetic background of sample you selected.
d) Optional parameters are much important for different sample, if the default parameters do not work well, you can change the number. SNP window size was suggested as 10~50, SNP step size was suggested as 5~20.

3. Donwload


a) After jobs finish, you can download all the results by a zip package.
b) If the figure not very well, then run mapping again with other parameters.
